Mackerel-Twilio HandsOn

  • Mackerelハンズオン

    • ハンズオン概要
    • オーガニゼーションを作成する
    • エージェントをインストールする
    • サービス／ロールでホスト管理をする
    • 監視ルールを追加する

    • 公式チェックプラグインを導入する

  • Twilioとの連携

    • Twilio連携ハンズオン概要
    • Twilioアカウントを作成する
    • Twilioから電話番号を取得する
    • Twilioと連携して架電通知をする
    • TwiMLエンドポイントを利用して架電通知をする
    • Twili番号をリリース（あとかたづけ）
      • 購入した電話番号をリリース

# Twilioの料金体系

Twilioはあらかじめアカウントにチャージしたクレジットが使用量に引き落とされる従量課金体系となっています。

ただし、電話番号に関しては回線費用が月額でかかるため、ハンズオン終了後に不要になる場合は忘れずに番号をリリースします。

# 購入した電話番号をリリース

Twilio電話番号コンソール (opens new window)を開き一覧から不要となった電話番号をクリックします。

詳細画面の最下部に表示されているRelease this Numberをクリックし不要になった番号をリリースします。

今回利用したTwiML Binsについては資料作成時点（2021年6月）で無料となっています。
